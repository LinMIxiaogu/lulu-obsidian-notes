 128.最长连续序列

### 题目说明

给定一个未排序的整数数组 `nums`，找出数字连续的最长序列长度。

要求设计时间复杂度为 `O(n)` 的算法。

示例：

```text
输入: nums = [100,4,200,1,3,2]
输出: 4
解释: 最长连续序列是 [1,2,3,4]，长度为 4
```

### Java 解答

```java
import java.util.HashSet;

class Solution {
    public int longestConsecutive(int[] nums) {
        HashSet<Integer> set = new HashSet<>();
        for (int num : nums) set.add(num);

        int maxLen = 0;
        for (int num : set) {
            if (!set.contains(num - 1)) {
                int index = num + 1;
                while (set.contains(index)) index++;
                maxLen = Math.max(maxLen, index - num);
            }
        }
        return maxLen;
    }
}
```

### Python 解答

```python
from typing import List


class Solution:
    def longestConsecutive(self, nums: List[int]) -> int:
        ans = 0
        st = set(nums)
        for x in st:
            if x - 1 in st:
                continue
            y = x + 1
            while y in st:
                y += 1
            ans = max(ans, y - x)
        return ans
```
