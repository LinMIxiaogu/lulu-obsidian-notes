 17.电话号码的字母组合

### 题目说明

给定一个仅包含数字 `2-9` 的字符串，返回所有它能表示的字母组合。

答案可以按任意顺序返回。

给出数字到字母的映射如下，与电话按键相同。注意 `1` 不对应任何字母。

示例：

```text
输入: digits = "23"
输出: ["ad","ae","af","bd","be","bf","cd","ce","cf"]
```

### Java 解答

```java
import java.util.ArrayList;
import java.util.HashMap;
import java.util.List;

class Solution {
    HashMap<Character, String> letterMap = new HashMap<>();
    List<String> res = new ArrayList<>();

    public List<String> letterCombinations(String digits) {
        letterMap.put('2', "abc");
        letterMap.put('3', "def");
        letterMap.put('4', "ghi");
        letterMap.put('5', "jkl");
        letterMap.put('6', "mno");
        letterMap.put('7', "pqrs");
        letterMap.put('8', "tuv");
        letterMap.put('9', "wxyz");
        if (digits.length() > 0) backTrack(digits, 0, "");
        return res;
    }

    private void backTrack(String digits, int index, String str) {
        if (index >= digits.length()) {
            res.add(str);
            return;
        }
        String valueStr = letterMap.get(digits.charAt(index));
        for (int i = 0; i < valueStr.length(); i++) {
            backTrack(digits, index + 1, str + valueStr.charAt(i));
        }
    }
}
```
