 5.最长回文子串
### 题目说明

给你一个字符串 `s`，找到 `s` 中最长的回文子串。

示例：
```text
输入: s = "babad"
输出: "bab"
解释: "aba" 同样是符合题意的答案。
```

### Java 解答

```java
class Solution {
    public String longestPalindrome(String s) {
        int n = s.length();
        int maxLen = 0;
        String res = "";
        boolean[][] dp = new boolean[n][n];
        for (int i = n - 1; i >= 0; i--) {
            for (int j = i; j < n; j++) {
                if (s.charAt(i) == s.charAt(j)) {
                    if (i == j || j - i == 1 || dp[i + 1][j - 1]) {
                        dp[i][j] = true;
                        if (j - i + 1 > maxLen) {
                            maxLen = j - i + 1;
                            res = s.substring(i, j + 1);
                        }
                    }
                }
            }
        }
        return res;
    }
}
```

### Python 解答

```python
class Solution:
    def longestPalindrome(self, s: str) -> str:
        n = len(s)
        dp = [[0] * n for i in range(n)]
        ans = ""
        for i in range(n - 1, -1, -1):
            for j in range(i, n):
                if s[i] == s[j]:
                    if j - i <= 1:
                        dp[i][j] = j - i + 1
                    elif dp[i + 1][j - 1] > 0:
                        dp[i][j] = dp[i + 1][j - 1] + 2

                if dp[i][j] > len(ans):
                    ans = s[i:j + 1]
        return ans
```
