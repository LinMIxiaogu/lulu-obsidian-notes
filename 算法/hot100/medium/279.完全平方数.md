 279.完全平方数
### 题目说明

给你一个整数 `n`，返回和为 `n` 的完全平方数的最少数量。

完全平方数是一个整数，其值等于另一个整数自乘的积。换句话说，其值等于一个整数和它自身的乘积。例如，`1`、`4`、`9` 和 `16` 都是完全平方数，而 `3` 和 `11` 不是。

示例：
```text
输入: n = 12
输出: 3
解释: 12 = 4 + 4 + 4
```

### Java 解答

```java
class Solution {
    public int numSquares(int n) {
        int[] dp = new int[n + 1];
        for (int i = 1; i <= n; i++) dp[i] = i;

        for (int i = 3; i <= n; i++) {
            for (int j = 2; j * j <= i; j++) {
                dp[i] = Math.min(dp[i], dp[i - j * j] + 1);
            }
        }
        return dp[n];
    }
}
```

### Python 解答

```python
class Solution:
    def numSquares(self, n: int) -> int:
        dp = [0] * (n + 1)
        for i in range(n + 1):
            dp[i] = i
            for j in range(2, i):
                if j * j > i:
                    break
                dp[i] = min(dp[i], dp[i - j * j] + 1)
        return dp[n]
```
