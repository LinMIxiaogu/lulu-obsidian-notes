 152.乘积最大子数组
### 题目说明

给你一个整数数组 `nums`，请你找出数组中乘积最大的非空连续子数组，并返回该子数组所对应的乘积。

测试用例的答案是一个 32 位整数。

示例：
```text
输入: nums = [2,3,-2,4]
输出: 6
解释: 子数组 [2,3] 有最大乘积 6。
```

### Java 解答

```java
class Solution {
    public int maxProduct(int[] nums) {
        int[][] multiSum = new int[nums.length][2];
        multiSum[0][0] = nums[0];
        multiSum[0][1] = nums[0];
        int res = nums[0];
        for (int i = 1; i < nums.length; i++) {
            multiSum[i][0] = Math.max(Math.max(multiSum[i - 1][0] * nums[i], multiSum[i - 1][1] * nums[i]), nums[i]);
            multiSum[i][1] = Math.min(Math.min(multiSum[i - 1][0] * nums[i], multiSum[i - 1][1] * nums[i]), nums[i]);
            res = Math.max(multiSum[i][0], res);
        }
        return res;
    }
}
```

### Python 解答

```python
class Solution:
    def maxProduct(self, nums: List[int]) -> int:
        pre_max = [1] * len(nums)
        pre_min = [1] * len(nums)
        pre_max[0] = pre_min[0] = nums[0]
        ans = nums[0]
        for i in range(1, len(nums)):
            pre_max[i] = max(nums[i], max(pre_max[i - 1] * nums[i], pre_min[i - 1] * nums[i]))
            pre_min[i] = min(nums[i], min(pre_max[i - 1] * nums[i], pre_min[i - 1] * nums[i]))
            ans = max(ans, pre_max[i])
        return ans
```
