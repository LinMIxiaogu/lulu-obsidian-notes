 169.多数元素
### 题目说明

给定一个大小为 `n` 的数组 `nums`，返回其中的多数元素。

多数元素是指在数组中出现次数大于 `⌊n / 2⌋` 的元素。

你可以假设数组是非空的，并且给定的数组总是存在多数元素。

示例：
```text
输入: nums = [3,2,3]
输出: 3
```

### Java 解答

```java
class Solution {
    public int majorityElement(int[] nums) {
        int res = nums[0];
        int count = 1;
        for (int i = 1; i < nums.length; i++) {
            if (count == 0) res = nums[i];
            if (nums[i] == res) {
                count++;
            } else {
                count--;
            }
        }
        return res;
    }
}
```
