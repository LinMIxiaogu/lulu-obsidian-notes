
> [!tip] 简单
 121.买卖股票的最佳时机

### 题目说明

给定一个数组 `prices`，它的第 `i` 个元素 `prices[i]` 表示一支给定股票第 `i` 天的价格。

你只能选择某一天买入这只股票，并选择在未来的某一个不同的日子卖出该股票。

设计一个算法来计算你所能获取的最大利润。

如果你不能获取任何利润，返回 `0`。

示例：

```text
输入: prices = [7,1,5,3,6,4]
输出: 5
解释: 在第 2 天买入，在第 5 天卖出，最大利润 = 6 - 1 = 5
```

### Java 解答

```java
class Solution {
    public int maxProfit(int[] prices) {
        int min = prices[0];
        int maxIncome = 0;
        for (int i = 1; i < prices.length; i++) {
            maxIncome = Math.max(maxIncome, prices[i] - min);
            min = Math.min(min, prices[i]);
        }

        return maxIncome;
    }
}
```
