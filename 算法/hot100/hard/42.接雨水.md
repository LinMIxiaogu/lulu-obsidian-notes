
> [!danger] 困难
 42.接雨水

### 题目说明

给定 `n` 个非负整数表示每个宽度为 `1` 的柱子的高度图，计算按此排列的柱子，下雨之后能接多少雨水。

示例：

```text
输入: height = [0,1,0,2,1,0,1,3,2,1,2,1]
输出: 6
```

### Java 解答

```java
class Solution {
    public int trap(int[] height) {
        int i = 0, j = height.length - 1;
        int leftMax = height[i];
        int rightMax = height[j];
        int resWater = 0;
        while (i < j) {
            leftMax = Math.max(leftMax, height[i]);
            rightMax = Math.max(rightMax, height[j]);
            if (leftMax < rightMax) {
                resWater += leftMax - height[i];
                i++;
            } else {
                resWater += rightMax - height[j];
                j--;
            }
        }
        return resWater;
    }
}
```

### Python 解答

```python
from typing import List


class Solution:
    def trap(self, height: List[int]) -> int:
        l, r = 0, len(height) - 1
        maxWater = 0
        leftCeil, rightCeil = 0, 0
        while l <= r:
            leftCeil = max(leftCeil, height[l])
            rightCeil = max(rightCeil, height[r])

            if leftCeil < rightCeil:
                maxWater += leftCeil - height[l]
                l += 1
            else:
                maxWater += rightCeil - height[r]
                r -= 1
        return maxWater
```
