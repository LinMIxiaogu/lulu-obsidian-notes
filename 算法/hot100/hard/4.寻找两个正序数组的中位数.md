 4.寻找两个正序数组的中位数

### 题目说明

给定两个大小分别为 `m` 和 `n` 的正序数组 `nums1` 和 `nums2`，请你找出并返回这两个正序数组的中位数。

算法的时间复杂度应该为 `O(log (m+n))`。

示例：

```text
输入: nums1 = [1,3], nums2 = [2]
输出: 2.00000
```

### Java 解答

```java
class Solution {
    public double findMedianSortedArrays(int[] nums1, int[] nums2) {
        int len1 = nums1.length;
        int len2 = nums2.length;
        int left = (len1 + len2 + 1) / 2;
        int right = (len1 + len2 + 2) / 2;

        int leftRes = findKth(nums1, 0, nums1.length - 1, nums2, 0, nums2.length - 1, left);
        int rightRes = findKth(nums1, 0, nums1.length - 1, nums2, 0, nums2.length - 1, right);
        return (leftRes + rightRes) * 0.5;
    }

    private int findKth(int[] nums1, int l1, int r1, int[] nums2, int l2, int r2, int k) {
        int len1 = r1 - l1 + 1;
        int len2 = r2 - l2 + 1;

        if (len1 > len2) return findKth(nums2, l2, r2, nums1, l1, r1, k);
        if (len1 == 0) return nums2[l2 + k - 1];
        if (k == 1) return Math.min(nums1[l1], nums2[l2]);

        int i = Math.min(l1 + (k / 2) - 1, l1 + len1 - 1);
        int j = Math.min(l2 + (k / 2) - 1, l2 + len2 - 1);

        if (nums1[i] > nums2[j]) {
            return findKth(nums1, l1, r1, nums2, j + 1, r2, k - (j - l2 + 1));
        } else {
            return findKth(nums1, i + 1, r1, nums2, l2, r2, k - (i - l1 + 1));
        }
    }
}
```
