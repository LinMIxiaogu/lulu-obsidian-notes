 295.数据流的中位数

### 题目说明

中位数是有序整数列表中的中间值。

- 如果列表的大小是奇数，中位数就是中间的数
- 如果列表的大小是偶数，中位数就是中间两个数的平均值

例如：

- `[2,3,4]` 的中位数是 `3`
- `[2,3]` 的中位数是 `(2 + 3) / 2 = 2.5`

请你实现 `MedianFinder` 类：

- `MedianFinder()` 初始化 `MedianFinder` 对象
- `void addNum(int num)` 将数据流中的整数 `num` 添加到数据结构中
- `double findMedian()` 返回到目前为止所有元素的中位数

示例：

```text
输入:
["MedianFinder", "addNum", "addNum", "findMedian", "addNum", "findMedian"]
[[], [1], [2], [], [3], []]
输出:
[null, null, null, 1.5, null, 2.0]
```

### Java 解答

```java
import java.util.PriorityQueue;

class MedianFinder {

    PriorityQueue<Integer> minQueue;
    PriorityQueue<Integer> maxQueue;

    public MedianFinder() {
        minQueue = new PriorityQueue<>(
            (o1, o2) -> o1 - o2
        );
        maxQueue = new PriorityQueue<>(
            (o1, o2) -> o2 - o1
        );
    }

    public void addNum(int num) {
        if (!minQueue.isEmpty() && num > minQueue.peek()) {
            minQueue.add(num);
            if (minQueue.size() > maxQueue.size()) {
                int data = minQueue.poll();
                maxQueue.add(data);
            }
        } else {
            maxQueue.add(num);
            if (maxQueue.size() - 1 > minQueue.size()) {
                int data = maxQueue.poll();
                minQueue.add(data);
            }
        }
    }

    public double findMedian() {
        if (maxQueue.size() > minQueue.size()) {
            return maxQueue.peek();
        } else {
            return (minQueue.peek() + maxQueue.peek()) * 0.5;
        }
    }
}

/**
 * Your MedianFinder object will be instantiated and called as such:
 * MedianFinder obj = new MedianFinder();
 * obj.addNum(num);
 * double param_2 = obj.findMedian();
 */
```
