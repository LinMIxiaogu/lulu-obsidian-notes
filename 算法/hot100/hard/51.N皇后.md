 51.N皇后

### 题目说明

按照国际象棋的规则，皇后可以攻击与之处在同一行、同一列或同一斜线上的棋子。

`n 皇后问题` 研究的是如何将 `n` 个皇后放置在 `n × n` 的棋盘上，并且使皇后彼此之间不能相互攻击。

给你一个整数 `n`，返回所有不同的 `n 皇后问题` 的解决方案。

每一种解法包含一个不同的 `n 皇后` 棋子放置方案，其中 `'Q'` 和 `'.'` 分别代表了皇后和空位。

示例：

```text
输入: n = 4
输出: [[".Q..","...Q","Q...","..Q."],["..Q.","Q...","...Q",".Q.."]]
```

### Java 解答

```java
import java.util.ArrayList;
import java.util.Arrays;
import java.util.List;

class Solution {
    List<List<String>> res = new ArrayList<>();
    List<Integer> col = new ArrayList<>();
    List<Integer> main = new ArrayList<>();
    List<Integer> sub = new ArrayList<>();

    public List<List<String>> solveNQueens(int n) {
        int[] queen = new int[n];
        backTrack(0, queen);
        return res;
    }

    private void backTrack(int row, int[] queen) {
        if (row == queen.length) {
            List<String> board = generator(queen);
            res.add(board);
            return;
        }
        for (int i = 0; i < queen.length; i++) {
            if (col.contains(i)) continue;
            if (main.contains(row - i)) continue;
            if (sub.contains(row + i)) continue;

            col.add(i);
            main.add(row - i);
            sub.add(row + i);

            queen[row] = i;
            backTrack(row + 1, queen);

            col.remove(col.size() - 1);
            main.remove(main.size() - 1);
            sub.remove(sub.size() - 1);
        }
    }

    private List<String> generator(int[] queen) {
        List<String> board = new ArrayList<>();
        for (int i = 0; i < queen.length; i++) {
            char[] row = new char[queen.length];
            Arrays.fill(row, '.');
            row[queen[i]] = 'Q';
            board.add(new String(row));
        }
        return board;
    }
}
```
