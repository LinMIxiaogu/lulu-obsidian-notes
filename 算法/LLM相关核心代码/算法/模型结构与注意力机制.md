# 模型结构与注意力机制

这类题目考查你对 Transformer 架构的理解深度。

## 核心考点

- **高频题：** 自注意力机制（Self-Attention）或多头注意力（Multi-Head Attention）的实现。
- **考核点：** 矩阵形状变化、Softmax 的应用、缩放因子的原因。
- **公式参考：**

$$Attention(Q, K, V) = \text{softmax}\left(\frac{QK^T}{\sqrt{d_k}}\right)V$$

- **建议：** 必须能用 PyTorch 或 NumPy 熟练写出张量相乘和维度转置的过程。

## 手写 Self-Attention

**背诵要点：** 重点在 $Q, K, V$ 的生成、缩放点积（Scaled Dot-Product）以及 Softmax 的维度。

```python
import torch
import torch.nn.functional as F

def self_attention(Q, K, V, mask=None):
    # d_k 是特征维度，用于缩放防止梯度消失
    d_k = Q.size(-1)

    # 1. 计算注意力分数: (batch, n, d_k) @ (batch, d_k, n) -> (batch, n, n)
    scores = torch.matmul(Q, K.transpose(-2, -1)) / torch.sqrt(torch.tensor(d_k))

    # 2. 如果有 mask (如 Decoder 中的 causal mask)，将对应位置设为负无穷
    if mask is not None:
        scores = scores.masked_fill(mask == 0, -1e9)

    # 3. Softmax 归一化得到权重
    attention_weights = F.softmax(scores, dim=-1)

    # 4. 加权求和得到输出: (batch, n, n) @ (batch, n, d_v) -> (batch, n, d_v)
    output = torch.matmul(attention_weights, V)

    return output, attention_weights
```
