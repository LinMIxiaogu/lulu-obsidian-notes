# 优化与微调

这类题目常见于算法岗和工程岗，重点在于理解参数高效微调的核心思想。

## 核心考点

- **高频题：** 手写 LoRA 的前向传播过程。
- **考核点：** 理解 $W = W_0 + \Delta W = W_0 + BA$，其中 $A$ 和 $B$ 是低秩矩阵。
- **建议：** 重点理解为什么 LoRA 能减少参数量，并能用简洁代码表达旁路矩阵的相加。

## LoRA 的前向传播实现

**背诵要点：** 原始权重不动，增加两个低秩矩阵 $A$ 和 $B$，输出相加。

```python
import torch.nn as nn

class LoRALayer(nn.Module):
    def __init__(self, in_dim, out_dim, rank=8, alpha=16):
        super().__init__()
        # 1. A 矩阵通常用高斯分布初始化
        self.lora_A = nn.Parameter(torch.randn(in_dim, rank))
        # 2. B 矩阵初始化为全 0，确保训练开始时 LoRA 旁路输出为 0
        self.lora_B = nn.Parameter(torch.zeros(rank, out_dim))
        # 3. 缩放系数
        self.scaling = alpha / rank

    def forward(self, x, original_weight_output):
        # original_weight_output 是原模型的线性层输出 (W0 * x)
        # lora_output = x @ A @ B
        lora_output = (x @ self.lora_A @ self.lora_B) * self.scaling
        return original_weight_output + lora_output
```
