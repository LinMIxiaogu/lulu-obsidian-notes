# LibAI 行程规划系统 — 项目概览

> **一句话描述：** LibAI 是一个 AI 驱动的智能行程规划系统，基于 LangGraph 多节点工作流，将用户的旅游需求转化为包含路线、酒店、交通、天气的完整行程方案，并以流式 SSE 事件推送给前端。

---

## 技术栈全景

```
┌─────────────────────────────────────────────────────┐
│                      前端/客户端                       │
│         SSE 流式接收 / WebSocket (语音)                │
└──────────────────────┬──────────────────────────────┘
                       │ HTTP / SSE
┌──────────────────────▼──────────────────────────────┐
│              FastAPI + Uvicorn  (ASGI)               │
│         TraceMiddleware (链路追踪 qtraceid)            │
└──────────────────────┬──────────────────────────────┘
                       │
┌──────────────────────▼──────────────────────────────┐
│              LangGraph 多节点工作流                    │
│  Reception → Thinker → Sector → Planner → Output    │
└──────┬─────────────┬──────────────┬─────────────────┘
       │             │              │
  LLM调用        外部API集成      数据存储
  (DeepSeek/     (携程/地图/      (PostgreSQL
   OpenAI)        天气/搜索)       + Redis)
```

| 层次         | 技术选型                                   | 说明               |
| ---------- | -------------------------------------- | ---------------- |
| **Web 框架** | FastAPI 0.115 + Uvicorn 0.34           | 异步 ASGI，支持流式响应   |
| **AI 编排**  | LangGraph 0.4.7                        | 有状态多节点 Agent 工作流 |
| **LLM 框架** | LangChain 0.3.25                       | LLM 调用统一封装       |
| **LLM 模型** | DeepSeek / OpenAI 兼容接口                 | 可通过配置切换          |
| **数据验证**   | Pydantic v2                            | 请求/响应对象          |
| **数据库**    | PostgreSQL (psycopg3) + SQLAlchemy 2.0 | 主存储              |
| **缓存**     | Redis                                  | 会话、路线结果缓存        |
| **爬虫**     | BeautifulSoup4 + Readability + Jina    | 小红书笔记内容提取        |
| **监控**     | Langfuse + OpenTelemetry + LangSmith   | LLM 链路追踪         |
| **日志**     | Loguru                                 | 结构化日志            |
| **实时通信**   | WebSocket + WebRTC VAD                 | 语音输入             |

## 核心功能一览

```
LibAI 能做什么？
├── 📍 智能行程规划      — 输入目的地/日期，自动生成多日路线
├── 🏨 酒店推荐/咨询     — 自然语言查酒店，支持条件过滤
├── ✈️  机票/火车票查询   — 查询价格、时刻、低价日历
├── 🌤️  天气查询          — 旅行日期天气预报
├── 📝 小红书笔记导入    — 从笔记链接直接生成行程
├── 🛣️  路线调整          — 对已生成路线进行修改
├── 📸 打卡机位生成      — 为景点推荐最佳拍照位置
└── 🎤 语音交互          — 支持语音输入的对话规划
```
