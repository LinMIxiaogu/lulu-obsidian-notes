## 项目名称
**GUI Agent 酒店智能助手（比价 & 笔记）**
---

## 项目描述
面向移动端用户的 GUI Agent 系统，通过 AI 编排层规划任务并下发操作指令，Android 前端借助无障碍服务自动驾驶手机跨平台（携程 / 美团 / 去哪儿）完成酒店比价与信息提取，最终汇总生成比价报告。

---
## 技术栈
`Python` · `LangGraph` · `LangChain` · `FastAPI` · `多模态 VLM` · `Redis` · `Android (Kotlin / Java)` · `AccessibilityService`

---
## 技术亮点

- 基于 LangGraph 实现 Plan-React 多 Agent 编排，支持按业务场景动态构建不同拓扑的工作流 DAG

- 设计前端驱动的分段式 Graph 执行模式，每轮请求仅执行单节点后返回，前端完成真实操作后携带新截图发起下一轮，以 Redis 跨请求累积状态，解决 LLM 推理与设备操作异步分离的问题

- 基于 Android 无障碍服务 实现 GUI Agent 执行层，接收 LLM 指令完成跨 APP 自动操作；同时通过 AccessibilityTree 元素查找实现权限引导自动化

- 通过 SSIM 截图相似度检测识别 Agent 死循环，结合 AOP 切面任务拦截与 LLM 主动触发的 Human-in-the-loop 机制，保障 Agent 可控性
